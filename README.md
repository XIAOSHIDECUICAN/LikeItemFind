# LikeItemFind Mod

一个用于在游戏地图上标记高价值物品和愿望单物品的 Unity Mod。

## 功能特性

### 🎯 核心功能

- **自动标记高价值物品**：自动标记价值超过 10,000 的物品
- **愿望单物品标记**：标记手动添加到愿望单的物品
- **双位置检测**：同时检测地面掉落物和箱子内的物品
- **实时地图显示**：在地图上显示圆形标记区域

### 🗺️ 地图集成

- 自动检测地图打开/关闭状态
- 使用游戏内置的地图标记系统
- 支持多场景物品标记
- 自动清理无效标记

### ⚡ 性能优化

- 场景加载时预缓存物品数据
- 智能对象生命周期管理
- 事件驱动的更新机制
